# Étapes pour installer WSL

1. Connectez votre ordinateur Windows à internet
2. Ouvrez un terminal Windows
3. Tapez `wsl`, puis Enter
4. Appuyez sur Enter pour accepter l'installation
5. Redémarrez votre ordinateur

## Pour démarrer votre nouveau sous-système Linux

1. Ouvrez un terminal Windows
2. Tapez `wsl --install Debian` (ou une autre distribution de votre choix), puis Enter
2. Tapez `wsl`, puis Enter
